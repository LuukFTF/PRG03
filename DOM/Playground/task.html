Je hebt nu de kennis in huis om elementen aan te maken, uit te lezen en informatie te overschrijven. Er zijn verschillende functies in Javascript om hier leuke dingen mee te doen. Als je meer diepgang zoekt in werking van DOM manipulatie, kijk dan hier.

De document.querySelector() is erg populair omdat je hiermee CSS selectors rechtstreeks in je Javascript selectie kunt gebruiken. Als je zeker weet dat je iets puur op basis van ID wilt hebben is het logisch om document.getElementById(); te gebruiken. Er is hierin geen goed of fout.

Je kunt nu wat gaan spelen met de geleverde HTML en Javascript binnen je browser om voor jezelf te kijken of je de geleerde functies onder controle hebt.

PS: is het je trouwens opgevallen dat het inladen van je Javascript bestand in je HTML altijd onderaan (voor sluiten van de <body> tag) gebeurd? Het is belangrijk dat je dit hier doet om te zorgen dat alle code al bestaat VOOR je Javascript gaat aanroepen met verwijzingen naar die elementen. Plaats je Javascript dus NIET in je <head>.
